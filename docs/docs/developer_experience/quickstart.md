# Quickstart

:::info Pre-requisites
Make sure you have [installed the Mason CLI](https://docs.brickhub.dev/installing) on your machine.
:::

## Create a new project with Jolt

``` sh title="Terminal"
# Add the Jolt brick if this is your first time using Jolt
mason add jolt

# Create a new project with Jolt
mason make jolt
```
